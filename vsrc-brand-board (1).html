// 01 the mark

The -/ monogram

The slash and dash lifted straight out of the wordmark: a path separator and a code fragment at once. Four constructions, differing only in how the dash meets the slash. The mark has to hold up shrunk to a favicon, printed with no red, and inverted every way, so the deciding question is never which is prettiest but which stays one legible shape when everything else is stripped away.

A — kerned apartHonest typographic reading. A clear gap holds the dash and slash as two separate marks.
B — joined at the footOne continuous stroke, but the foot join reads as a check mark and loses the dash.

C — sharing a terminalThe dash meets the slash's flank at one node. Connected, yet still plainly a dash beside a slash.

V-/Src, set in Instrument Serif

V is we; Src is source. The name states the business: we, source. Set in the display serif, the whole problem is the -/ cluster, the hinge of the lockup, where a raw hyphen and slash sit too loose. The fragment is tracked in tight so it binds to the letters instead of floating between them.

// 04 palette

One accent. Two reds. No cool greys.

Warm blacks, warm creams, and a single accent, signal red at hue 28. There is no blue, no green, no purple, not even for success. The red means recording, live, important: the thing that stops the scroll, so it stays rare enough to keep meaning that.

Same hue, two jobs. --primary is the fill that carries cream text safely; --signal is the vivid display red, and cream on it is 3.47:1, which is exactly why the second shade has to exist rather than being an interchangeable tint. As text, the legal red flips by theme: on near-black use --signal (5.0:1); on cream paper small red text must use --primary (5.3:1), because --signal on cream is only 4.4:1.
